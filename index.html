Graceless Degradation

Giving IE 6 browsers the experience they demand!

With all this talk of graceful browser degradation, who is speaking for the other side?

Graceless Degradation is a WordPress plugin that gives visitors still using Internet Explorer 6 an appropriate user experience.

Here's your expected, graceful WordPress experience using Mac OS X and Google's Chrome browser:

Here's the same page using Windows XP with Internet Explorer 6:

  • Comic Sans fonts everywhere!
  • Emphasis tags (bold, italic, strong, and underline) blink! Even though IE doesn't support the <blink> tag!
  • <hr> tags become animated rainbows! They're not moving in the 2-frame animated gif above, but trust me, they're animated.
  • Animated mailbox graphics added to "mailto:" links!

Installation is simple!

Just download the plugin, unpack it under wp-content/plugins, and activate the plugin in your WordPress console.

That's it! Graceless Degradation has no moving parts or sharp edges.
